# 첫 번째 글: 블로그를 시작하며

프레임워크 없이 순수 HTML, CSS, JavaScript만으로 만든 마크다운 블로그입니다.

## 어떻게 동작하나요?

이 블로그는 마크다운 파일을 브라우저에서 직접 읽어 HTML로 변환합니다. [marked.js](https://marked.js.org/)를 사용해 마크다운을 파싱하고, 별도의 빌드 과정 없이 실시간으로 렌더링합니다.

## 주요 기능

- **다크 모드** — 시스템 설정을 자동 감지하고, 수동 토글도 가능합니다
- **반응형 디자인** — 모바일과 데스크톱 모두에서 편하게 읽을 수 있습니다
- **깔끔한 타이포그래피** — 읽기에 집중할 수 있는 미니멀한 디자인

## 코드 예시

```javascript
const response = await fetch('posts/posts.json');
const posts = await response.json();
```

## 인용문

> 단순함은 궁극의 정교함이다.
> — 레오나르도 다 빈치

## 마무리

새 글을 추가하려면 `posts/` 폴더에 마크다운 파일을 만들고 `posts.json`에 메타데이터를 추가하면 됩니다.
